# 电脑笔记

## 电脑文件共享手机

cmd输入ipconfig查看ip地址，
cmd输入python -m http.server 888
其中888可自定义任意数字
手机同一个网络下，输入ip ：  888

例如ip = 192.168.1.3
则共享网址为：http://192.168.1.3:888/
